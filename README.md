### Prerequisites
- Python 3.8+

### Installation

```bash
# Clone the repository

cd aruvikolam

# Set up environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

```bash

uvicorn app:app --reload
```

Visit `http://localhost:8000` in your browser to access the application.
